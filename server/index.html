Treinamento Angular

Bem vindo ao backend do Treinamento Angular!

Essa pequena aplicação (super simples e praticamente sem validações)
foi feita com nodejs e um banco super simples que salva os dados em arquivos JSON.

Autenticação

  • POST /v1/auth/login - Autentica o usuário e retorna um token para as proximas requisições
    (usuário padrão está no arquivo database/users.json).
  • POST /v1/auth/register - Cria um novo usuário.
  • GET /v1/auth/unique-email - Verifica se o e-mail informado é unico no sistema.

Compras (Nescessário enviar token de autenticação)

  • GET /v1/purchases - Lista todas as compras do usuário logado.
  • POST /v1/purchases - Cria uma nova compra.
  • POST /v1/purchases/{id} - Edita um compra existente.
  • DELETE /v1/purchases/{id} - Exclui uma compra existente.
